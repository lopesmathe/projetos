Nessa aula vamos ver como utilizar o atributo Class

De uma forma tecnica

Define quais classes CSS serao aplicados no elemento

De uma forma informal

Quando pegaos duas tegs <p> e queremos que ela fique com a mesmas propiedas CSS utilizamos o class

Na pratica

        
<style>
            .paragrafo{
                color: blue;
                font-size: 2em;
            } 
</style>


body
<p class="paragrafo">
Hi my name is Matheus
</p>
<p class="paragrafo">
Hi my name not is Matheus
</p>
/body
